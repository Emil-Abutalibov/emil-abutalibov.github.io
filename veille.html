Introduction

Qu'est-ce que la veille technologique ?

La veille technologique est une activité qui consiste à s'informer très souvent sur les nouveautés d'un sujet technique. Ce n'est pas juste lire pour s'amuser, c'est une méthode pour rester compétitif et efficace dans son travail.

J'ai choisi l'IA Générative car c'est un domaine qui bouge tous les jours dans l'informatique. Il est important de savoir comment ces outils évoluent pour mieux programmer et protéger les données.

SchémaPourquoi j'ai choisi l'IA Générative pour mon BTS.

Ma Méthodologie de Veille

🎯
Étape 1

Le choix des thèmes précis

Pour que ma veille soit utile, je ne regarde pas "tout" sur l'IA, car il y a trop d'informations. J'ai divisé ma recherche en trois parties :

  • Le code assisté par l'IA : Je surveille les outils comme GitHub Copilot, Claude, ChatGPT, Gemini pour voir comment ils aident à écrire du code.
  • La sécurité informatique : Je regarde si les IA peuvent créer des virus ou si elles volent des informations privées.
  • L'actualité générale : Je suis les grandes annonces des entreprises comme OpenAI ou Google pour connaître les nouveaux modèles.

Étape 2

La sélection des meilleures sources (Curation)

Pour avoir des informations de qualité, j'ai choisi des sources sérieuses. C'est l'étape de la curation : on choisit ce qui est bon et on jette ce qui est inutile.

  • Réseaux Sociaux : Je suis des experts sur Twitter (X) qui partagent leurs travaux et découvertes.
  • Blogs de développeurs : Je lis des articles écrits par des professionnels sur des sites comme Octo Talks ou Développez.com.
  • Sites officiels : Je consulte les sites des technologies que je suis pour lire les journaux de mise à jour.

⚙️
Étape 3

L'utilisation d'outils automatiques (Agrégation)

Pour ne pas perdre de temps, j'utilise un agrégateur de flux. C'est un logiciel qui va chercher les articles tout seul sur mes sites préférés et les rassemble au même endroit.

  • Outil utilisé : J'utilise Feedly pour centraliser mes lectures.
  • Organisation : J'ai créé des dossiers pour trier les articles par thèmes automatiquement.
  • Gain de temps : Je peux voir en un coup d'œil les 10 nouvelles informations les plus importantes du jour.
Outil techniqueMon interface Feedly avec mes flux RSS organisés.
💾
Étape 4

Enregistrement et Partage des connaissances

Une veille est inutile si on oublie tout le lendemain. Il faut stocker l'information et la partager avec les autres.

  • Stockage : J'utilise Pocket pour enregistrer les liens les plus intéressants pour les relire plus tard.
  • Reformulation : Pour bien comprendre, j'essaie d'expliquer l'article avec mes propres phrases.
  • Partage : Je communique mes découvertes sur Reddit pour aider la communauté informatique.

🚀
Étape 5

Mise en pratique (Tests réels)

La veille doit finir par une mise en pratique pour tester ce qu'on a lu. L'auto-formation est une partie importante de mon apprentissage en BTS.

  • Tests de code : J'ai utilisé des outils comme Claude pour m'aider à comprendre des algorithmes complexes.
  • Comparaison : J'ai testé plusieurs IA pour voir laquelle écrit le meilleur code Java ou Python.
